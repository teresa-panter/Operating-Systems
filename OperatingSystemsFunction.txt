One of the operating system functions is memory managment. This is just one of the many things an operating system does. An operating system is also the software that performs the basic tasks. 
Another function that an operating system does is file management. This is the software that manages your data files in your computer. It has limited space and is designed to manage individual or group files, like documents.
Another thing that an operating system does is help with things you want to print. It is the software that links your system to a printer.
